# 説明

授業で使うリポジトリ。

簡単に Laravel6 の開発環境ができる。

-   PHP8 系
-   Laravel6 系
-   MySQL8.0
-   phpmyadmin

## 使い方

### STEP1

以下のコマンドを実行

```bash
# 注意1:けっこう時間がかかりますが仕様です。
# 注意2:ローカルサーバー（MAMPなど）を全部停止しないとエラーがでます。
$ git clone https://github.com/tatun55/gs-lab11-laravel6
$ cd gs-lab11-laravel6
$ make create-project
```

### STEP2

ブラウザでアクセス

```
http://localhost
http://localhost:8080
```
